Project Overview

This website was designed and deployed as part of my AWS Cloud project. It uses Amazon S3 for storage, CloudFront for global CDN distribution, and Route53 for DNS management with HTTPS enabled via AWS Certificate Manager.

Key Highlights:

  • ✅ 99.9% uptime using AWS Cloud infrastructure
  • ✅ 40% faster page load time via CloudFront caching
  • ✅ Automated deployment using AWS CLI & CodePipeline
